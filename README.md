# yuzuwatch-x

Post daily picture to [@yuzuwatch](https://x.com/yuzuwatch) X account.

## Usage

```sh
go run main.go -ih example.com -ck <CONSUMER_KEY> -cs <CONSUMER_SECRET> -at <ACCESS_TOKEN> -as <ACCESS_SECRET>
```
